## Quest Description

Explorers require reliable supplies for dangerous territories.

## Objectives

- **Deliver:** [Specific Supply] to quest location (varies daily)
- **Quantity:** [Specific Amount] (varies between 10-50 based on daily quest)

### Daily Quest Details
When this quest appears, it will specify:
- **One specific supply** from the possible list below
- **Exact quantity** needed (within the 10-50 range)
- **Delivery location** (same as pickup location)

### Possible Supplies
- [Rope](/docs/resources/individual/rope), [Minor Defense Potion](/docs/items/individual/minor_defense_potion), [Defense Potion](/docs/items/individual/defense_potion), [Greater Defense Potion](/docs/items/individual/greater_defense_potion)

## Rewards

Upon completion, you will receive:

- **Gold:** 25-100% of the base gold value of items (skewed toward lower end, rounded to nearest 20)
- **Reputation XP:** 25% of gold reward (rounded to nearest 10)  
- **Fishing XP:** 50% of XP normally gained from obtaining the items (rounded to nearest 50)

## Found At

Any Job Board Region
