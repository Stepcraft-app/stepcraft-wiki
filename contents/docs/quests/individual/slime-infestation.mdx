## Quest Description

The underground passages are overrun with hostile slimes that threaten the mining operations.

## Objectives

- **Deliver:** [Specific Slime Type] to quest location (varies daily)
- **Quantity:** [Specific Amount] (varies between 10-100 based on daily quest)

### Daily Quest Details
When this quest appears, it will specify:
- **One specific slime type** from the possible list below
- **Exact quantity** needed (within the 10-100 range)
- **Delivery location** (same as pickup location)

### Possible Slime Types
- [Green Slime](/docs/resources/individual/green_slime), [Blue Slime](/docs/resources/individual/blue_slime), [Red Slime](/docs/resources/individual/red_slime), [Yellow Slime](/docs/resources/individual/yellow_slime)

## Rewards

Upon completion, you will receive:

- **Gold:** 25-100% of the base gold value of items (skewed toward lower end, rounded to nearest 20)
- **Reputation XP:** 25% of gold reward (rounded to nearest 10)  
- **Combat XP:** 50% of XP normally gained from obtaining the items (rounded to nearest 50)

## Found At

Any Job Board Region
