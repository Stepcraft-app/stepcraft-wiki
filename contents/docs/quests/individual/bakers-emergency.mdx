## Quest Description

The village baker has run out of essential ingredients.

## Objectives

- **Deliver:** [Specific Ingredient] to quest location (varies daily)
- **Quantity:** [Specific Amount] (varies between 10-100 based on daily quest)

### Daily Quest Details
When this quest appears, it will specify:
- **One specific baking ingredient** from the possible list below
- **Exact quantity** needed (within the 10-100 range)
- **Delivery location** (same as pickup location)

### Possible Baking Ingredients
- [Wheat](/docs/resources/individual/wheat), [Flour](/docs/resources/individual/flour), [Sugar](/docs/resources/individual/sugar), [Egg](/docs/resources/individual/egg), [Milk](/docs/resources/individual/milk)

## Rewards

Upon completion, you will receive:

- **Gold:** 25-100% of the base gold value of items (skewed toward lower end, rounded to nearest 20)
- **Reputation XP:** 25% of gold reward (rounded to nearest 10)  
- **Farming XP:** 50% of XP normally gained from obtaining the items (rounded to nearest 50)

## Found At

Any Job Board Region
