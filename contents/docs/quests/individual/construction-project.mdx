## Quest Description

A major construction project requires quality wooden planks to continue.

## Objectives

- **Deliver:** [Specific Plank Type] to quest location (varies daily)
- **Quantity:** [Specific Amount] (varies between 25-100 based on daily quest)

### Daily Quest Details
When this quest appears, it will specify:
- **One specific plank type** from the possible list below
- **Exact quantity** needed (within the 25-100 range)
- **Delivery location** (same as pickup location)

### Possible Plank Types
- [Birch Plank](/docs/resources/individual/birch_plank), [Oak Plank](/docs/resources/individual/oak_plank), [Pine Plank](/docs/resources/individual/pine_plank), [Redwood Plank](/docs/resources/individual/redwood_plank), [Silverleaf Plank](/docs/resources/individual/silverleaf_plank), [Eldritchwood Plank](/docs/resources/individual/eldritchwood_plank)

## Rewards

Upon completion, you will receive:

- **Gold:** 25-100% of the base gold value of items (skewed toward lower end, rounded to nearest 20)
- **Reputation XP:** 25% of gold reward (rounded to nearest 10)  
- **Carpentry XP:** 50% of XP normally gained from obtaining the items (rounded to nearest 50)

## Found At

Any Job Board Region
