## ğŸ“– Writing Guidelines

### Content Style
- **Be clear and concise** - explain things simply
- **Stay accurate** - base information on current game version
- **Be helpful** - think about what players need to know
- **Link related content** - connect pages together

### Formatting Tips
Every page should start with this structure:

```markdown
---
title: "Your Page Title"
description: "Brief description for search results"
---

# Your Page Title

Your content here...

## Main Sections
Use ## for major sections

### Subsections  
Use ### for smaller sections
```

### Using Tables
Great for stats, recipes, and structured data:

```markdown
| Item | Level | Damage |
|------|-------|--------|
| Iron Sword | 10 | 25 |
| Steel Sword | 20 | 45 |
```

### Adding Links
Link to other wiki pages:
```markdown
[Iron Ore](/docs/resources/individual/iron_ore)
[Mining Skill](/docs/skills/individual/mining)
```
